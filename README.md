## Setup via Terminal
```
$ git clone https://github.com/OsmiumNetwork/EduKit-New
$ cd EduKit-New
```
> Skip the first two steps if you deployed via one of the buttons above
```
$ npm i
$ cd Ultraviolet-Static
$ npm i
$ cd ..
$ npm i ./Ultraviolet-Static
$ npm start
```
